# Changelog

## [1.0.1] - 2025-10-13

### Added
- 更新日表示機能を追加（`show_modified`パラメータ）
- 投稿情報セクションに更新日を表示できるように実装
- ウィザードUIに更新日チェックボックスを追加

### Fixed
- ウィザードで生成されるPHPコードのカルーセル表示で最後の1つが空白になる問題を修正
- カルーセルHTMLに`data-effective-cols`と`data-post-count`属性を追加

## [1.0.0] - 2025-09-14

### Added
- 初回リリース
- グリッド、リスト、カルーセルの3つのデザインパターンを実装
- ビジュアルウィザード機能 - コード不要でショートコードを生成
- 投稿タイプ選択機能（標準投稿、固定ページ、カスタム投稿タイプ）
- カテゴリ・タグによるフィルタリング機能
- 高度なカスタマイズオプション（列数、行数、色、配置）
- レスポンシブデザイン対応（モバイル、タブレット、デスクトップ）
- ページネーション機能（数字、前後ナビ、もっと読むボタン）
- 表示要素の個別制御（タイトル、日付、著者、カテゴリ、タグ、抜粋）
- SEO最適化機能（タイトルタグ選択：h1〜h6、div）
- 並び替え機能（日付、タイトル、ランダム、メニュー順）
- カルーセル自動再生機能
- 管理画面での詳細設定
- 多言語対応準備（Text Domain設定）

### Features
- **グリッドレイアウト**: 最大6列までの柔軟なグリッド表示
- **リストレイアウト**: サムネイル位置を選択可能なリスト表示
- **カルーセル**: タッチ対応の滑らかなカルーセル表示
- **ビジュアルウィザード**: 初心者でも簡単にショートコードを生成
- **プレビュー機能**: リアルタイムでデザインを確認
- **レスポンシブ**: すべてのデバイスで最適化された表示

### Technical
- WordPress 5.0以上対応
- PHP 7.2以上対応
- jQuery依存（WordPressバンドル版を使用）
- オブジェクト指向設計による保守性の高いコード構造
- WordPress Coding Standardsに準拠
- セキュリティ対策（nonce、エスケープ、サニタイズ）
